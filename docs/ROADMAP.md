# Roadmap (після веб-MVP)

## Android (етап 1)
- **Ціль**: Android-клієнт для проходження програми та логування.
- **Підхід**: мобільний застосунок працює з тим самим API, що й веб.
- **Функції**:
  - логін/реєстрація
  - “сьогоднішній день”
  - перегляд дня + відмітка виконання
  - журнал
  - бібліотека вправ
