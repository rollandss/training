# Контент-план (українською)

## 1) Структура контенту для вправи
Для кожної вправи робимо однаковий шаблон:
- **Назва**
- **Навіщо** (1–2 речення)
- **Техніка** (короткі пункти)
- **Поширені помилки** (пункти)
- **Полегшення** (для новачків)
- **Ускладнення** (на майбутнє)
- **Коли зупинитись** (біль/дискомфорт, застереження)

## 2) Мінімальний список вправ (для MVP)
### Тяга / спина
- Австралійські підтягування (на низькому турніку/перекладині)
- Підтягування (негативи як полегшення)
- Виси/активація лопаток (скейпулар пул-ап)

### Жим / груди-плечі
- Віджимання від підлоги
- Віджимання з колін / від опори
- Віджимання на брусах (як опційне обладнання)

### Ноги
- Присідання з вагою тіла
- Випади (крок назад/вперед — обрати один варіант)

### Кор
- Планка (передня)
- Бічна планка
- Hollow body hold (або “човник” — обрати і стандартизувати)

### Розминка/мобільність
- Розігрів плечового поясу (круги, лопатки)
- Розминка зап’ясть (для віджимань)
- Розігрів стегон/гомілкостопу

### Заминка
- Легке розтягування грудних/широчайших/квадрицепсу
- Дихання 1–2 хв

## 3) Контентні сторінки (MVP)
- **Як проходити програму**
- **Розминка: навіщо і як**
- **Відновлення**: сон, вода, навантаження
- **Безпека**: біль vs “робоче відчуття”, коли робити паузу
- **FAQ**: “пропустив день”, “болять зап’ястя”, “болять лікті від турніка”, “нема часу”

## 4) Відео (після MVP, опційно)
- По 20–40 секунд на вправу:
  - 1 кадр “як правильно”
  - 1 кадр “типова помилка”
  - 1 підказка текстом

## 5) Процес наповнення
- Спочатку: **чернетки** (коротко, без ідеальної редактури).
- Потім: **уніфікація термінів** (перекласти однаково в усіх сторінках).
- Потім: **редактура** під читання з телефону.
