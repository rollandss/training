# Чекліст запуску MVP

## 1) Продуктові рішення

- **Тривалість програми**: 100 днів
- **Ритм**: “тренування/відпочинок” (зафіксувати шаблон календаря)
- **Обладнання MVP**: турнік (обов’язковий)
- **Рівні**: 1 рівень (найпростіший) або 2 рівні (новачок/середній)
- **Стиль контенту**: коротко (bullet points) чи розгорнуто (абзаци)
- **Тон**: спортивний/дружній/суворий (обрати)

## 2) UX та екрани

- **Лендінг**: офер, “як працює”, FAQ, CTA “Почати”
- **Реєстрація/логін**: email + пароль, відновлення пароля
- **Онбординг**: рівень, обладнання, обмеження (плечі/коліна/спина), вибір програми
- **Дашборд**: “сьогодні”, прогрес тижня, швидкі посилання
- **Сторінка дня**: список вправ, схема підходів/повторів, таймер (опційно), “виконано”
- **Журнал**: календар або список, фільтри “виконано/пропущено”
- **Бібліотека вправ**: картка вправи (техніка/помилки/полегшення)
- **Профіль**: зміна пароля, вихід
- **Адмінка**: вправи, програми, дні, порядок вправ

## 3) Контент (українською)

- **Список вправ (10–20)** визначений
- **Для кожної вправи**: назва, цільові м’язи, техніка, помилки, полегшення/ускладнення, “коли зупинитись”
- **Розминка/заминка**: базові шаблони
- **Безпека**: дисклеймер + рекомендації при болі
- **Політики**: приватність, умови, дисклеймер відповідальності
- **100-денна структура** описана: фази + шаблони днів + правила прогресії

## 4) Дані та API (мінімально)

- **Модель даних** затверджена (`users`, `programs`, `days`, `exercises`, `logs`)
- **API endpoints** затверджені (auth, program, today, log)
- **Ролі**: user/admin

## 5) Технічна реалізація

- **DB**: Postgres налаштований
- **ORM**: міграції і seed (мінімальний)
- **Auth**: реєстрація/логін/вихід, захист сторінок
- **Admin**: доступ тільки адмінам
- **Валідації**: форми, помилки, edge cases
- **Адаптивність**: перевірка на мобільному

## 6) Якість та реліз

- **Smoke-test сценарій**: новий юзер → онбординг → день 1 → виконано → журнал
- **SEO**: тайтли, описи, og:image (мінімально)
- **Аналітика** (опційно): реєстрації, старт програми, виконання дня
- **Деплой**: staging + production
